# Südsalat App

Projektordner für die Companion-App zum Podcast "Südsalat" von Thorsten & Jenny.

Homepage: https://www.südsalat.eu
RSS-Feed: https://www.südsalat.eu/podcast.rss

## Ordnerstruktur

- `01-Brainstorming/` — Konzept, Entscheidungen, offene Fragen
- `02-Design/` — Design-System (Farben, Schriften, Layout-Werte), Logo & Branding-Assets
- `03-Backend/` — PHP/MySQL-API + Admin-Bereich (läuft auf Strato)
- `04-App/` — Flutter-App (iOS & Android)
- `05-Assets/` — Logo, Bilder, sonstige Rohdateien

Stand: 2026-07-12 — siehe `01-Brainstorming/Konzept.md` für den aktuellen Planungsstand.
